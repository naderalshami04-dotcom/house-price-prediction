# 🏠 House Price Prediction

This project predicts house prices using Machine Learning.

## 📌 Features
- Area
- Total Rooms
- Furnishing Status

## 🤖 Model
- Random Forest Regressor

## 📊 Performance
- R²: 0.53
- MAE: 0.23
